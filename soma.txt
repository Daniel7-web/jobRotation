1 - Observe o trecho de código abaixo:

int INDICE = 13, SOMA = 0, K = 0; 

enquanto K < INDICE faça 

{ 
  k = k + 1;  SOMA = SOMA + K; 
} 

imprimir(SOMA);    

. Qual será o valor da variável SOMA?

O valor da variável SOMA será a soma dos números inteiros de 1 a 13, ou seja:

SOMA = 1 + 2 + 3 + ... + 12 + 13

Podemos calcular essa soma usando a fórmula da soma dos n primeiros números inteiros:

SOMA = (n * (n + 1)) / 2

Substituindo n por 13, temos:

SOMA = (13 * (13 + 1)) / 2 = 91

Portanto, o valor da variável SOMA será igual a 91.
